# Text To Video Generator

> Generate animated video from text description using PIL and MoviePy

## Overview
This project is part of a comprehensive AI/ML portfolio covering generative ai python.

## Features
- Production-ready implementation
- Well-documented code with examples
- Easy to customize and extend

## Installation

```bash
git clone https://github.com/jaychampaneri14/text-to-video-generator.git
cd text-to-video-generator
pip install -r requirements.txt
python main.py
```

## Requirements
See `requirements.txt` for full dependencies.
Main libraries: Generative Ai, Python, Video

## Usage
```bash
python main.py
```
Run with `--help` for full options.
